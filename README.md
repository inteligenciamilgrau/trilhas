# Trilhas de Cursos — Inteligência Mil Grau

Landing page estática com as trilhas de cursos (playlists) do canal
[Inteligência Mil Grau](https://www.youtube.com/) no YouTube.

Cada trilha pode ser aberta diretamente no YouTube ou expandida na própria
página para ver a lista de vídeos individuais. Os dados são atualizados
automaticamente uma vez por dia via GitHub Actions.

Hospedado como site estático no GitHub Pages.
